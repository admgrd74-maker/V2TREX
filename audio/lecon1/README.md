# Dossier audio — lecon 1

Dépose ici tes fichiers .mp3 générés avec ElevenLabs, un par réplique :
01.mp3, 02.mp3, 03.mp3, ...

Dans une leçon, le champ "audio" peut pointer vers ces fichiers, par ex :
"audio": "../audio/lecon1/01.mp3"

(Si tu utilises l'admin avec export tout-en-un, les audios sont déjà intégrés
au fichier .mtlesson et ce dossier n'est pas indispensable.)
